The beauty of this framework is that most of these javascript plugins need very little javascript manipulation. Some just need a small tag of activation in the footer while others are completely in tact as long as you use the correct classes.

Most of these functions are controlled by 'data' tags being added to hyperlinks.

Collapse

This element allows us to give similar information in a sortable way. Users can select exactly what they want to read without being sent to a new page to read about it.

Standard Collapse 
  <div class="accordion" id="accordion2">
  <div class="accordion-group">
    <div class="accordion-heading">
      <a class="accordion-toggle" data-toggle="collapse" data-parent="#accordion2" href="#collapseOne">
        Collapsible Group Item #1
      </a>
    </div>
    <div id="collapseOne" class="accordion-body collapse in">
      <div class="accordion-inner">
        Anim pariatur cliche...
      </div>
    </div>
  </div>
  <div class="accordion-group">
    <div class="accordion-heading">
      <a class="accordion-toggle" data-toggle="collapse" data-parent="#accordion2" href="#collapseTwo">
        Collapsible Group Item #2
      </a>
    </div>
    <div id="collapseTwo" class="accordion-body collapse">
      <div class="accordion-inner">
        Anim pariatur cliche...
      </div>
    </div>
  </div>
</div>

Affix

The side navigation bar is an example of the Affix plugin. It allows objects to track with the scroll bar. This is useful for keeping navigation accessible no matter where the user has scrolled.

These tags need to be added to an element to affix the element:

  <div data-spy="affix" data-offset-top="200">...</div>

There are a couple of other settings such as affix-top and affix-bottom that can be used as well.

The offset number is the number of pixels that the user must scroll to have the affix kick in. Some manipulation of the CSS might be needed to make the affixed element function correctly.
